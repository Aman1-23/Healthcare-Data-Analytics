# 🏥 Healthcare Data Analytics Project

## 📌 Overview

This project focuses on analyzing healthcare data to extract meaningful insights that support better decision-making in hospitals and healthcare organizations. The analysis is performed on 10,000+ patient records using SQL, Python, Excel, and Power BI.

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Matplotlib)
* **SQL**
* **Power BI**
* **Excel**

---

## 📊 Key Features

* Data cleaning and preprocessing of healthcare dataset
* Exploratory Data Analysis (EDA) using Python
* SQL queries for extracting insights
* Interactive Power BI dashboard with KPIs and visualizations

---

## 📈 Key Insights

* Identified disease-wise patient distribution
* Analyzed hospital-wise revenue generation
* Calculated average billing and patient stay duration
* Improved reporting efficiency and decision-making

---

## 📊 Dashboard

The Power BI dashboard includes:

* Total Patients
* Total Revenue
* Average Billing
* Average Stay Duration
* Disease Distribution
* Hospital Revenue Analysis
* Gender-wise Patient Distribution
* Monthly Admission Trends
